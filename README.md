# `secretary`

Python scaffolding framework.

# Usage
```bash
$ ./secretary.py --new project-name --template cmake
```

# Templates
- make (C)
	- Toolchain: Make
- cmake (C++ 20)
	- Toolchain: g++, ninja, cmake, clang-format, ccache
- conan (C++ 20)
	- Dependencies: g++, ninja, cmake, conan clang-format, ccache
